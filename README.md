# Trade Analyzer

**What it does:**
- Upload your trade history CSV from any broker and get an instant analysis
- See your true returns after fees, slippage, bid-ask spreads, and taxes
- Compare your performance against SPY and QQQ over the exact same time period
- Get plain-English feedback on your trading habits — no finance background needed

## How to run

```bash
# Terminal 1 — frontend
cd frontend
npm run dev

# Terminal 2 — backend
cd backend
flask run
```

Navigate to http://localhost:5173/

## Stack

- **Frontend:** React + Vite + Tailwind
- **Backend:** Python + Flask
